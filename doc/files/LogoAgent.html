LogoAgent

Basic Agent for Logo models.

Arguments

  • execute: A function that gets the agent itself as argument and is called every time step.
  • init: A function that gets the agent itself as argument and will be executed when the agent is initialized.

Usage

import("logo")

LogoAgent{}

Functions

breed  Reproduce the Agent to a random neighbor Cell.
countNeighbors  Count the number of Agents in the neighbor cells.
emptyNeighbor  Return an empty neighbor Cell.
relocate  Walk to a random neighbor Cell.

breed 

Reproduce the Agent to a random neighbor Cell. If there is no empty neighbor cell then this function returns nil.

Arguments

  • #1: An Agent.

Usage

import("logo")

agent = LogoAgent{
   execute = function(self)
       self:breed()
       self:relocate()
   end
}

countNeighbors 

Count the number of Agents in the neighbor cells.

Arguments

  • #1: An Agent.

Usage

import("logo")

agent = LogoAgent{
   execute = function(self)
       print(self:countNeighbors())
   end
}

emptyNeighbor 

Return an empty neighbor Cell. If there is no empty neighbor then it returns false.

Arguments

  • #1: An Agent.

Usage

import("logo")

agent = LogoAgent{
   execute = function(self)
       if not self:emptyNeighbor() then
           print("not empty")
       end
   end
}

relocate 

Walk to a random neighbor Cell. If there is no empty neighbor then this function returns false.

Arguments

Usage

import("logo")

agent = LogoAgent{
   execute = function(self)
       self:relocate()
